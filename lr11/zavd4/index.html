Завдання 4 - Вкладки (Tabs)

HTML

HTML (HyperText Markup Language) — це мова розмітки для створення веб-сторінок. Вона визначає структуру та зміст сторінки за допомогою тегів.

Основні теги: <html>, <head>, <body>, <div>, <p>, <a>.

CSS

CSS (Cascading Style Sheets) — це мова стилів, яка використовується для опису зовнішнього вигляду HTML-документів. CSS дозволяє змінювати кольори, шрифти, відступи та розташування елементів.

Приклад: color: red; font-size: 16px;

JavaScript

JavaScript — це мова програмування, яка дозволяє створювати інтерактивні веб-сторінки. З її допомогою можна обробляти події, змінювати DOM, робити запити до серверу тощо.

Приклад: document.getElementById('id').innerHTML = 'Hello';

Bootstrap

Bootstrap — це CSS-фреймворк для швидкої розробки адаптивних веб-сторінок. Він включає готову сітку, компоненти (кнопки, форми, модальні вікна, каруселі) та утиліти для стилізації.

Поточна версія: Bootstrap 5.3
