Bootstrap сторінка

Bootstrap — це популярний CSS-фреймворк для створення адаптивних веб-сторінок. Він містить готові компоненти, сітку та утиліти для швидкої верстки.

Ця сторінка використовує Bootstrap 5 підключений через CDN. Навігаційна панель має випадаюче підменю та адаптується під мобільні пристрої.
